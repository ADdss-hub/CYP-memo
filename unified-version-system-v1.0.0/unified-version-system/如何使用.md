# 统一版本管理系统 - 使用说明

## 📦 这是什么？

这是一个**完整的、独立的**版本管理系统包，可以直接使用或分发。

---

## 🚀 使用方式

### 方式 1: 在当前项目中使用（已配置）

当前 CYP-memo 项目已经配置好了，直接使用：

```bash
# 在项目根目录运行
npm run version:patch
npm run version:minor
npm run version:major
npm run version:info
npm run version:validate
npm run version:history
```

### 方式 2: 全局安装

```bash
# 进入此目录
cd unified-version-system

# 全局链接
npm link

# 然后在任何项目中使用
uvm help
uvm patch
uvm info
```

### 方式 3: 复制到其他项目

```bash
# 复制整个目录到其他项目
cp -r unified-version-system /path/to/other/project/

# 在目标项目中使用
cd /path/to/other/project
node unified-version-system/bin/uvm.js help
```

### 方式 4: 发布到 NPM

```bash
# 进入此目录
cd unified-version-system

# 登录 NPM
npm login

# 发布
npm publish
```

### 方式 5: 打包分发

```bash
# 压缩整个目录
zip -r unified-version-system.zip unified-version-system/

# 或使用 tar
tar -czf unified-version-system.tar.gz unified-version-system/

# 分享给其他人
```
